# 지구다 API

## 1. ExtensionAPI
유튜브 영상을 보며 등장하는 용어설명과 내용 요약을 담당


## 2. PreprocessAPI
유튜브 스크립트의 오탈자, 띄어쓰기와 같은 전처리
<br>
지식 구조화를 위한 문장 구문화 (BERT 모델 활용)
<br>
구문화 이후 Neo4j GraphDB 업데이트


## 3. DetectionAPI
하나의 토픽 내의 구문들 간의 숨은 관계를 탐지
<br>
R-VGAE 모델 활용
